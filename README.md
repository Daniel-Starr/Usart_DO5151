# STM32U575 × DO5151 GNSS驯服恒温晶振 串口透传工程

基于 **STM32U575VGT6**（Nucleo 板）实现与 **TechOne DO5151 GNSS驯服恒温晶振** 的双向串口透传，用户可通过电脑串口助手直接向晶振发送控制指令并接收响应数据。

## 硬件平台

| 组件 | 型号 / 规格 |
|------|-----------|
| MCU | STM32U575VGT6（LQFP100，Cortex-M33） |
| 晶振模块 | TechOne DO5151 GNSS驯服恒温晶振（10MHz CMOS输出） |
| 开发板 | NUCLEO-U575 系列 |
| 调试器 | 板载 ST-Link |

## 功能说明

- **双向透传**：电脑 ↔ USART1 ↔ MCU ↔ USART2 ↔ DO5151 晶振
- **中断接收 + 环形缓冲区**：USART1 和 USART2 均采用中断接收模式，通过 256 字节环形缓冲区避免在 115200 波特率下丢字节
- **指令交互**：支持 DO5151 全部控制指令（`$DID?*`、`$RST*`、`$UPE1*` 等）
- **状态监测**：可接收晶振每秒上报的 UPV 状态帧和 TOD 授时帧

## 串口配置

| 参数 | USART1（电脑） | USART2（晶振） |
|------|---------------|---------------|
| 波特率 | 115200 | 115200 |
| 数据位 | 8 | 8 |
| 停止位 | 1 | 1 |
| 校验位 | 无 | 无 |
| 流控 | 无 | 无 |

## 引脚连接

```
STM32U575                DO5151
──────────              ──────────
PA9  (USART1_TX) ───→  电脑（通过 ST-Link VCP）
PA10 (USART1_RX) ←───  电脑（通过 ST-Link VCP）
PA2  (USART2_TX) ───→  Pin 6 (RX)
PA3  (USART2_RX) ←───  Pin 7 (TX)
GND              ───→  Pin 4 / Pin 13 (GND)
```

> ⚠️ DO5151 电源为 5V（Pin 3），信号电平为 3.3V，可直接与 STM32 连接无需电平转换。

## 快速开始

1. 使用 STM32CubeIDE 或 Keil 打开工程
2. 编译并烧录到 NUCLEO-U575 开发板
3. 按照引脚连接表接好 DO5151 晶振
4. 打开串口助手，选择 ST-Link 虚拟串口（115200, 8N1）
5. 发送 `$DID?*\r\n` 查询设备信息，应收到类似：
   ```
   $OK,DID-DO5151-1.02-10-SQUARE-00176*
   ```

## 常用 DO5151 指令

| 指令 | 功能 | 返回示例 |
|------|------|---------|
| `$DID?*\r\n` | 查询设备标识 | `$OK,DID-DO5151-1.02-10-SQUARE-00176*` |
| `$RST*\r\n` | 复位设备 | `$OK,RST*` |
| `$CLK?*\r\n` | 查询输出频率 | `$OK,CLK10*` |
| `$UPE1*\r\n` | 开启状态帧上报（每秒） | `$OK,UPE1*` |
| `$UPE0*\r\n` | 关闭状态帧上报 | `$OK,UPE0*` |
| `$PPP?*\r\n` | 查询相差测量结果 | `$OK,PPP4.321*` |
| `$DEN?*\r\n` | 查询驯服使能状态 | `$OK,DEN1*` |

## UPV 状态帧解析

晶振每秒上报的状态帧格式：

```
$UPV1,0,0,0,0,0,+0.000,+2.64487527,+0.000E+00,+0.000E+00,+0.000E+00,556,0,0,0*
```

| 字段 | 含义 |
|------|------|
| 1 | 驯服使能（1=使能） |
| 2 | 参考秒接入（0=未接入，1=接入） |
| 3 | 频差粗测完成标志 |
| 4 | 秒脉冲同步标志 |
| 5 | 驯服状态（0=未驯服，1=驯服中，2=锁定，3=守时） |
| 6 | 驯服模式（0=自动，1=压控电压） |
| 7 | 相差测量结果（ns） |
| 8 | 当前压控电压（V） |
| 9 | 百秒平均准确度 |
| 10 | 千秒平均准确度 |
| 11 | 天平均准确度 |
| 12 | 无参考秒累计时长（s） |
| 13 | 参考秒接入时长（s） |
| 14 | 驯服时长（s） |
| 15 | 守时时长（s） |
